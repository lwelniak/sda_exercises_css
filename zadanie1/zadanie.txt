1. Stwórz paragraf, ustaw dla niego rozmiar czcionki na 12px
2. Stwórz drugi paragraf i ustaw dla niego rozmiar czcionki na 20px i grubość na 700;
3. Do pierwszego paragrafu dodaj klasę, i ustaw dla niego rozmiar czcionki na 16px używając tej klasy.
4. Stwórz dwa paragrafy, oraz jeden element span. Niech każdy ma wpisany przykładowy tekst. Ustaw dla wszystkich zielone tło.
5. Stwórz element div, wpisz mu przykładową treść, nadaj mu indentyfikator oraz klasę. Następnie ustaw mu poniższe style:
	
	#identyfikator_diva {
		background: yellow;
	}

	div {
		background: red;
	}

	.klasa_diva {
		background: blue;
	}

6. Dla diva z poprzedniego zadania, wymuś niebieski kolor tła przy użyciu słowa kluczowego !important.
